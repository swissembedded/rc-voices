# Notes on voice + licensing

- **Engine:** Kokoro-82M (https://github.com/hexgrad/kokoro), Apache-2.0.
  Preset voices ship with the model; generated audio is clean to use and
  redistribute, including commercially.
- **We do not clone anyone.** The pack is synthesized with a generic
  Kokoro preset chosen to fit a voice profile (lists/voice_target.md),
  not from any real person's recordings - no voice/personality-right
  questions.
- **Text lists** are derived from EdgeTX/OpenTX and INAV (mode names and
  the expected sound-file names). Those are functional identifiers /
  short data, used so the pack matches what each radio plays.
- **Reference packs** are used only to confirm the expected filenames and
  for loudness matching, never as the voice, and are not redistributed
  from here.
